# Formation Complète

Parcours d'apprentissage complet couvrant Conda, VSCode et Git pour la géomatique et l'analyse de données spatiales.

## Structure du cours

La formation est organisée en trois modules principaux :

### 1. Conda - Gestion d'environnements Python
Apprenez à utiliser Conda pour gérer vos environnements Python et installer les librairies géospatiales essentielles.

- **[2.1 Théorie](2.1-conda-theorie.md)** : Concepts fondamentaux de Conda
- **[2.2 Installation](2.2-conda-installation.md)** : Installer Conda sur votre système
- **[2.3 Environnements](2.3-conda-environnements.md)** : Créer et gérer des environnements
- **[2.4 Librairies géospatiales](2.4-conda-librairies.md)** : Installation de GDAL, Geopandas, Rasterio, etc.
- **[2.5 Commandes essentielles](2.5-conda-commandes.md)** : Référence des commandes utiles

### 2. VSCode - Éditeur pour la géomatique
Configurez VSCode pour le développement Python en géomatique et utilisez efficacement ses fonctionnalités.

- **[3.1 Interface](3.1-vscode-interface.md)** : Présentation de l'interface VSCode
- **[3.2 Extensions](3.2-vscode-extensions.md)** : Extensions essentielles pour la géomatique
- **[3.3 Terminal & Conda](3.3-vscode-terminal.md)** : Intégrer Conda dans le terminal VSCode
- **[3.4 Git & GitHub](3.4-git-github.md)** : Contrôle de version et collaboration
- **[3.5 Démarche complète](3.5-demarche-complete.md)** : Intégration complète de l'écosystème

### 3. Ressources et guides
- **[Présentation](0-presentation.md)** : Vue d'ensemble du projet
- **[Introduction](1-introduction.md)** : Avant de commencer
- **[4. Ressources](4-ressources.md)** : Fichiers, configurations et liens utiles

## Pour commencer

**Débutant ?** Suivez les sections dans l'ordre, en commençant par la présentation et l'introduction.

**Expérimenté ?** Consultez directement le module qui vous intéresse.
